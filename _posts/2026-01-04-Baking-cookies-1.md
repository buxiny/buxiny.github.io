## 1. 杏仁脆脆曲奇 (Almond Crispy Cookies)

![](https://img.cook1cook.com/upload/cover/201706/5/2705459507cbd40d945.jpg)

| 序号 | 配料名称 | 重量 (g) |
| ------ | ---------- | ---------- |
| 1    | 黄油     | 80g      |
| 2    | 糖粉     | 33g      |
| 3    | 蛋清液   | 15g      |
| 4    | 低筋粉   | 110g     |
| 5    | 杏仁粉   | 10g      |
| 6    | 细盐     | 1g       |
| 7    | 杏仁片   | 20g      |

**烘焙参数：**

- **风炉：**  150°C 烤 25 分钟 + 140°C 烤 5 分钟
- **最后：**  关火焖 5 分钟
- **方案(二)（成品上色较浅）：**  150°C 烤 15 分钟 + 140°C 烤 14 分钟，关火焖 5 分钟。

**TIPS（整形技巧）：**

- 整形成圆柱形，直径  6~6.5mm。
- 一份成型长度约为 22~23cm。

## 3. 燕麦坚果曲奇 (Oatmeal Nut Cookies)

![c1469d4a5d1a45439a3ad996846b34ff_1920w_2560h.jpg](https://s2.loli.net/2026/01/04/9jPU3YsXZ2O1LCz.jpg)

| 序号 | 配料名称 | 重量 (g) | 备注         |
| ------ | ---------- | ---------- | -------------- |
| 1    | 全蛋液   | 55g      | 约一个大鸡蛋 |
| 2    | 金砂糖   | 40g      |              |
| 3    | 海盐     | 1.0g     |              |
| 4    | 黄油     | 80g      |              |
| 5    | 低筋粉   | 90g      |              |
| 6    | 小苏打   | 1g       |              |
| 7    | 泡打粉   | 1g       |              |
| 8    | 生燕麦片 | 85-90g   |              |
| 9    | 蔓越莓干 | 40g      |              |
| 10   | 混合坚果 | 25g      |              |

**烘焙参数：**

- **风炉：**  150°C 烤 20 分钟
- **最后：**  关火焖 5 分钟
- **成品参考：**  面团分量约 20-21g/个，此配方约可制作 20 块。

**操作步骤 (TIPS)：**

1. 黄油溶化后加入金砂糖、海盐、全蛋液，搅拌均匀。
2. 筛入粉类（低粉+小苏打+泡打粉），加入燕麦片拌匀。
3. 蔓越莓干和坚果切碎粒后加入拌匀，最后放入冰箱冷藏 1 小时以上即可。

## 4. 蓝罐曲奇 (Kjeldsens Style Cookies)

| 序号 | 配料名称 | 重量 (g) | 备注             |
| ------ | ---------- | ---------- | ------------------ |
| 1    | 黄油     | 80g      |                  |
| 2    | 糖粉     | 32g      |                  |
| 3    | 蛋清     | 13g      | 也可用淡奶油替换 |
| 4    | 低筋粉   | 115g     |                  |
| 5    | 椰蓉     | 5g       |                  |
| 6    | 细盐     | 1g       |                  |
| 7    | 香草精   | 1g       |                  |
| 8    | 粗砂糖   | 少量     | 蘸表面用         |

**烘焙参数：**

- **方案 A：**  风炉 150°C 烤 25 分钟 + 140°C 烤 5 分钟，关火焖 5 分钟。
- **方案 B：**  150°C 烤 15 分钟 + 140°C 烤 14 分钟，关火焖 5 分钟。

**TIPS：**

- 整形成长方体条状（长度约 18cm），切正方形片状，厚度 8mm - 10mm。
